# 01-blink-led

This project demonstrates the simplest Arduino program: blinking an LED.  
It is the first step in hardware programming labs for students.

## Contents
- **code/**: Arduino sketch (`blink_led.ino`)
- **wiring/**: Breadboard and schematic diagrams
- **simulation/**: Tinkercad project link
- **docs/**: Lab guide with step-by-step instructions
- **bom.csv**: Bill of Materials

## How to Run
1. Connect the LED and resistor as shown in `wiring/`.
2. Open `code/blink_led.ino` in Arduino IDE.
3. Select **Arduino UNO** board and correct COM port.
4. Upload the code.
5. Observe the LED blinking every second.

## Educational Notes
- Introduces **digital output** using `digitalWrite()`.
- Demonstrates the use of `pinMode()` and `delay()`.
- Forms the foundation for more complex projects (inputs, sensors, PWM).

# 01-چشمک-زدن-LED

این پروژه ساده‌ترین برنامه آردوینو را نشان می‌دهد: چشمک زدن یک LED.  
این اولین گام در آزمایش‌های برنامه‌نویسی سخت‌افزار برای دانشجویان کاردانی مهندسی کامپیوتر است.

## محتوا
- **code/**: کد آردوینو (`blink_led.ino`)
- **wiring/**: دیاگرام‌های بردبرد و شماتیک
- **simulation/**: لینک پروژه در Tinkercad
- **docs/**: راهنمای آزمایش گام‌به‌گام
- **bom.csv**: لیست قطعات

## روش اجرا
1. LED و مقاومت را طبق شکل در پوشه `wiring/` وصل کنید.
2. فایل `blink_led.ino` را در Arduino IDE باز کنید.
3. برد **Arduino UNO** و پورت صحیح را انتخاب کنید.
4. کد را آپلود کنید.
5. مشاهده کنید که LED هر ثانیه روشن و خاموش می‌شود.
